# CodSoft Machine Learning Internship

This repository contains the Machine Learning internship tasks completed under CodSoft.

## Tasks Completed

### Task 1: Spam SMS Detection
Classifies SMS messages as Spam or Ham using TF-IDF Vectorization and Naive Bayes Classification.

### Task 2: Credit Card Fraud Detection
Detects fraudulent financial transactions using Machine Learning and Random Forest Classification.

### Task 3: Movie Genre Classification
Predicts movie genres from textual descriptions using Natural Language Processing (NLP) and Logistic Regression.
