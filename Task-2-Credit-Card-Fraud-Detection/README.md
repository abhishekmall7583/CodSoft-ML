# Credit Card Fraud Detection

## Description
Credit Card Fraud Detection is a Machine Learning project that identifies fraudulent transactions based on transaction details and customer information. The project uses data preprocessing, feature encoding, and Random Forest Classification to distinguish between genuine and fraudulent transactions.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-Learn
- Random Forest Classifier
- Google Colab

## Dataset Features
- Transaction Amount
- Merchant Information
- Customer Details
- Transaction Location
- Transaction Time
- Fraud Label (is_fraud)

## Methodology
1. Data Collection
2. Data Preprocessing
3. Feature Engineering
4. Label Encoding
5. Model Training
6. Fraud Detection
7. Performance Evaluation

## Result
The model successfully classifies transactions as fraudulent or genuine with high accuracy.

## Conclusion
This project demonstrates the application of Machine Learning techniques in detecting fraudulent financial transactions and improving transaction security.
